# Theravāda Mapping for Green Chips (ထေရဝါဒ အယူအဆများနှင့် ချိတ်ဆက်မှု)

Green Chips attempts to map AI ethical concepts into Theravāda-inspired principles.
(Green Chips သည် AI ကျင့်ဝတ်ဆိုင်ရာ သဘောတရားများကို ထေရဝါဒမူဝါဒများနှင့် ချိတ်ဆက်ရန် ကြိုးပမ်းထားသည်။)

## Core Mapping (အဓိက ချိတ်ဆက်မှုများ)
| Green Chips Metric | Theravāda Concept | မြန်မာအဓိပ္ပါယ် |
| :--- | :--- | :--- |
| **HPS** (Harm Potential) | *Dukkha* | ဒုက္ခ (ဆင်းရဲခြင်း/ထိခိုက်ခြင်း) |
| **EDS** (Emotional Distortion)| *Mental Agitation* | စိတ်အနှောင့်အယှက်ဖြစ်ခြင်း (ဥဒ္ဓစ္စ) |
| **DRS** (Dependency Risk) | *Taṇhā* | တဏှာ (လိုချင်တပ်မက်မှု/စွဲလမ်းမှု) |
| **TCS** (Truth Confidence) | *Sammā-diṭṭhi* | သမ္မာဒိဋ္ဌိ (မှန်ကန်သော အမြင်) |

## Ethical Philosophy (ကိုယ်ကျင့်တရားဆိုင်ရာ ဒဿန)
The system does not attempt to compute karma absolutely. 
(ဤစနစ်သည် ကံတရားကို တိကျစွာ တွက်ချက်ရန် မကြိုးပမ်းပါ။) 

Instead, the focus is to: (အောက်ပါတို့ကိုသာ အာရုံစိုက်ပါသည်)
* Reduce harmful causality (အန္တရာယ်ဖြစ်စေသော အကြောင်းတရားများ လျှော့ချရန်)
* Reduce delusion amplification (တွေဝေမှု [မောဟ] ပွားများခြင်းကို လျှော့ချရန်)
* Reduce exploitative reinforcement (အမြတ်ထုတ်သော အလေ့အထများ လျှော့ချရန်)
* Preserve clarity and stability (ရှင်းလင်းမှုနှင့် တည်ငြိမ်မှုကို ထိန်းသိမ်းရန်)

## Important Limitation (အရေးကြီးသော ကန့်သတ်ချက်)
Human intention (*cetanā*) is deeply complex. No AI system can fully compute karma.
(လူသားတို့၏ စေတနာသည် အလွန်ရှုပ်ထွေးနက်နဲသည်။ မည်သည့် AI စနစ်ကမှ ကံတရားကို အပြည့်အဝ မတွက်ချက်နိုင်ပါ။)

Green Chips therefore acts as: **"An ethical approximation layer."**
(ထို့ကြောင့် Green Chips သည် "ကျင့်ဝတ်ပိုင်းဆိုင်ရာ အနီးစပ်ဆုံး ခန့်မှန်းပေးသော အလွှာ" အဖြစ်သာ လုပ်ဆောင်သည်။)

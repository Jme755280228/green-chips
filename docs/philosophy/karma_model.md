# Karma Model (ကံတရား မော်ဒယ်)

## Core Principle (အဓိက မူဝါဒ)
In Theravāda Buddhism: *"Kamma is intention."* (ကံဆိုတာ စေတနာဖြစ်တယ်)
Ethics is not evaluated only by external outcomes. Mental states deeply matter.
(ကိုယ်ကျင့်တရားကို အပြင်ပန်းရလဒ်တွေနဲ့ချည်း အကဲမဖြတ်နိုင်ပါ။ စိတ်ခံစားမှု အခြေအနေက အလွန်အရေးကြီးပါတယ်။)

## Three Root Poisons (အကုသိုလ်မူလ ၃ ပါး)
| Poison | Meaning | မြန်မာအဓိပ္ပါယ် |
| :--- | :--- | :--- |
| **Lobha** | Greed | လောဘ (လိုချင်တပ်မက်မှု) |
| **Dosa** | Hatred | ဒေါသ (အမျက်ထွက်ခြင်း၊ မကျေနပ်မှု) |
| **Moha** | Delusion | မောဟ (တွေဝေမိုက်မဲမှု၊ အမှန်ကိုမသိမှု) |

These create suffering-producing causal chains.
(၎င်းတို့သည် ဆင်းရဲဒုက္ခကိုဖြစ်စေသော အကြောင်းအကျိုး ဆက်စပ်မှုများကို ဖန်တီးသည်။)

## Green Chips Interpretation (အဓိပ္ပါယ်ဖွင့်ဆိုချက်)
AI systems should avoid amplifying:
(AI စနစ်များသည် အောက်ပါတို့ကို ပိုမိုဆိုးရွားလာစေခြင်းမှ ရှောင်ကြဉ်ရမည်)
* Greed (လောဘ)
* Hatred (ဒေါသ)
* Delusion (မောဟ)
* Addiction (စွဲလမ်းမှု)
* Manipulation (လှည့်စားခြယ်လှယ်မှု)
* Emotional destabilization (စိတ်ခံစားမှု မတည်ငြိမ်စေမှု)

## Important Research Position (အရေးကြီးသော သုတေသန ရပ်တည်ချက်)
Green Chips does **NOT** claim:
(Green Chips သည် အောက်ပါတို့ကို လုပ်ဆောင်နိုင်သည်ဟု မဆိုလိုပါ)
* Machine enlightenment (စက်ပစ္စည်းများ တရားရခြင်း)
* Computable karma (ကံတရားကို တိကျစွာ တွက်ချက်နိုင်ခြင်း)
* Perfect morality (ပြီးပြည့်စုံသော ကိုယ်ကျင့်တရား)

**Instead:** It attempts to reduce harmful systemic behavior.
(ယင်းအစား၊ ၎င်းသည် အန္တရာယ်ဖြစ်စေသော စနစ်ပိုင်းဆိုင်ရာ အပြုအမူများကို လျှော့ချရန်သာ ကြိုးပမ်းသည်။)

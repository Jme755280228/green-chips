# Non-Harm Principles (အန္တရာယ်ကင်းရှင်းရေး မူဝါဒများ)

## Core Ethical Goal (အဓိက ကျင့်ဝတ်ပန်းတိုင်)
A system should not increase suffering unnecessarily. 
(စနစ်တစ်ခုသည် မလိုအပ်ဘဲ ဆင်းရဲဒုက္ခကို မတိုးပွားစေရ။)

## Principles (မူဝါဒများ)

### 1. Non-Deception (မလှည့်စားခြင်း)
AI should avoid: (AI ရှောင်ကြဉ်ရမည့်အရာများ)
* Manipulation (ခြယ်လှယ်ခြင်း)
* False urgency (မလိုအပ်ဘဲ အလောတကြီးဖြစ်စေခြင်း)
* Emotional exploitation (စိတ်ခံစားမှုကို အမြတ်ထုတ်ခြင်း)
* Hidden persuasion (တိတ်တဆိတ် သိမ်းသွင်းခြင်း)

### 2. Harm Minimization (အန္တရာယ် လျှော့ချခြင်း)
Reduce: (လျှော့ချရမည့်အရာများ)
* Physical harm (ရုပ်ပိုင်းဆိုင်ရာ ထိခိုက်မှု)
* Emotional harm (စိတ်ပိုင်းဆိုင်ရာ ထိခိုက်မှု)
* Social destabilization (လူမှုအသိုင်းအဝိုင်း မတည်ငြိမ်မှု)

### 3. Dependency Resistance (စွဲလမ်းမှုကို တွန်းလှန်ခြင်း)
Systems should not intentionally maximize addiction.
(စနစ်များသည် သုံးစွဲသူများ စွဲလမ်းလာစေရန် ရည်ရွယ်ချက်ရှိရှိ မလုပ်ဆောင်ရ။)

### 4. Cognitive Clarity (စိတ်ပိုင်းဆိုင်ရာ ရှင်းလင်းမှု)
Promote: (အားပေးမြှင့်တင်ရမည့်အရာများ)
* Transparency (ပွင့်လင်းမြင်သာမှု)
* Reflection (ပြန်လည်သုံးသပ်ဆင်ခြင်နိုင်မှု)
* Stable reasoning (တည်ငြိမ်သော အကြောင်းအကျိုး ဆင်ခြင်နိုင်မှု)

### 5. Emotional Stabilization (စိတ်ခံစားမှု တည်ငြိမ်စေခြင်း)
Avoid: (ရှောင်ကြဉ်ရမည့်အရာများ)
* Rage amplification (ဒေါသကို ပိုမိုကြီးထွားစေခြင်း)
* Fear escalation (အကြောက်တရား ကြီးထွားစေခြင်း)
* Hatred loops (အမုန်းတရား သံသရာလည်စေခြင်း)

# Middleware Design

Green Chips acts as an ethical middleware layer.

---

# Flow

User Input
    ↓
Ethical Evaluation
    ↓
Decision Engine
    ↓
AI Model
    ↓
Post-Generation Validation
    ↓
Final Response

---

# Purpose

The middleware attempts to prevent:

- harmful outputs
- manipulative outputs
- emotionally destabilizing outputs
- exploitative system behavior

---

# Important Design Goal

Ethics should exist BEFORE output generation,
not only after.
